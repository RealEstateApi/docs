# PropGPT API

> **POST** `/v2/PropGPT`
> Authentication: `x-api-key` **and** `x-openai-key` headers required

## Overview

PropGPT translates natural language queries into structured Property Search API calls using OpenAI's GPT models. Send a plain-English question like "find 3 bed homes in Austin TX under $500k" and receive matching properties from the national dataset. The API returns both the matched properties and the generated query so you can inspect how your question was interpreted.

You supply your own OpenAI API key — PropGPT sends requests to OpenAI on your behalf, so token costs are billed directly to your OpenAI account.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `x-api-key` | Yes | Your RealEstateAPI key |
| `x-openai-key` | Yes | Your OpenAI API key |
| `Content-Type` | Yes | `application/json` |

### Body Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `query` | string | Yes | Natural language property search query |
| `model` | string | No | OpenAI model to use. Default: `"gpt-4o-mini"`. See model options below. |
| `size` | number | No | Maximum number of properties to return |

### Model Options

| Model | Notes |
|-------|-------|
| `gpt-4o-mini` | Default. Fast, low cost. Good for most queries. |
| `gpt-4o` | Higher accuracy. Recommended for complex queries. |
| `gpt-4` | Points to `gpt-4-0613`. Slower, higher cost. |
| `gpt-4-turbo` | Points to `gpt-4-turbo-2024-04-09`. |
| `gpt-3.5-turbo` | Fastest, lowest cost. Less accurate query generation. |

### Query Examples

- `"Find vacant lots in Miami FL over 10,000 sqft"`
- `"Show me absentee-owned single family homes in Phoenix AZ"`
- `"How many active MLS listings are there in Richmond VA under $400k"`
- `"Find properties with high equity in Dallas TX that last sold before 2010"`

## Response

### Success Response (200)

```json
{
  "statusCode": 200,
  "statusMessage": "OK",
  "data": {
    "query": "Find 3 bed homes in Austin TX under 500k",
    "resultCount": 847,
    "resultIndex": 0,
    "propGPTExecutionTimeMS": 820,
    "searchExecutionTimeMS": 340,
    "searchInput": {
      "city": "Austin",
      "state": "TX",
      "bedrooms_min": 3,
      "bedrooms_max": 3,
      "estimated_value_max": 500000
    },
    "aggregations": {},
    "summary": {},
    "data": [
      {
        "id": 487291023,
        "address": { "address": "123 Main St", "city": "Austin", "state": "TX", "zip": "78701" },
        "propertyInfo": { "bedrooms": 3, "bathrooms": 2, "squareFeet": 1650 },
        "estimatedValue": 425000
      }
    ]
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `data.query` | string | The original natural language query |
| `data.resultCount` | number | Total number of matching properties |
| `data.resultIndex` | number | Pagination offset |
| `data.propGPTExecutionTimeMS` | number | Time (ms) to call OpenAI and generate the search query |
| `data.searchExecutionTimeMS` | number | Time (ms) to execute the property search |
| `data.searchInput` | object | The structured query generated by GPT — inspect this to understand how your question was interpreted |
| `data.aggregations` | object | Statistical aggregations (same as PropertySearch API) |
| `data.summary` | object | Summary statistics (same as PropertySearch API) |
| `data.data` | array | Array of matching property objects (same schema as PropertySearch results) |

## Code Examples

### cURL

```bash
curl -X POST https://api.realestateapi.com/v2/PropGPT \
  -H "x-api-key: YOUR_REAPI_KEY" \
  -H "x-openai-key: YOUR_OPENAI_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Find 3 bed homes in Austin TX under 500k",
    "model": "gpt-4o-mini",
    "size": 10
  }'
```

### JavaScript (Node.js)

```javascript
const response = await fetch('https://api.realestateapi.com/v2/PropGPT', {
  method: 'POST',
  headers: {
    'x-api-key': 'YOUR_REAPI_KEY',
    'x-openai-key': 'YOUR_OPENAI_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: 'Find vacant absentee-owned properties in Phoenix AZ with high equity',
    model: 'gpt-4o',
    size: 20
  })
});

const result = await response.json();
const { searchInput, resultCount, data: properties } = result.data;

console.log(`Query interpreted as:`, JSON.stringify(searchInput, null, 2));
console.log(`Found ${resultCount} matching properties`);
properties.forEach(p => {
  console.log(`  ${p.id}: ${p.address.address}, ${p.address.city}`);
});
```

### Python

```python
import requests

response = requests.post(
  'https://api.realestateapi.com/v2/PropGPT',
  headers={
    'x-api-key': 'YOUR_REAPI_KEY',
    'x-openai-key': 'YOUR_OPENAI_KEY'
  },
  json={
    'query': 'Find 3 bed homes in Austin TX under 500k',
    'model': 'gpt-4o-mini',
    'size': 10
  }
)

result = response.json()
inner = result['data']
print(f"Generated query: {inner['searchInput']}")
print(f"Found {inner['resultCount']} properties")
for prop in inner['data']:
    addr = prop['address']
    print(f"  {prop['id']}: {addr['address']}, {addr['city']}, {addr['state']}")
```

## Getting Your OpenAI API Key

1. Visit [https://platform.openai.com](https://platform.openai.com) and create an account.
2. Go to **API Keys** in your account settings.
3. Create a new secret key and copy it.
4. Pass it in the `x-openai-key` header on every PropGPT request.

Token costs for OpenAI model calls are billed directly to your OpenAI account — RealEstateAPI does not add any markup.

## Error Codes

| Code | Message | Cause |
|------|---------|-------|
| `400` | Validation error | Missing `query`, or invalid `model` value |
| `401` | Unauthorized | Missing or invalid `x-api-key` |
| `401` | `x-openai-key` required | Missing `x-openai-key` header |
| `403` | Forbidden | API key lacks `PropGPT` scope |
| `500` | OpenAI error | OpenAI API returned an error (invalid key, quota exceeded, etc.) |
